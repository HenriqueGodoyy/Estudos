Principais Formatações

Negrito / Destaque

Nesta frase, o termo está em negrito usando a tag strong (semântica) ou a tag b (não semântica).

Itálico / Ênfase

Nesta frase, o termo em itálico está usando a tag i (não semântica) e a tag em (semântica).

Observação

É recomendado não usar as tags <b> e <i> por não serem semânticas.

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno.

Texto Deletado

Podemos marcar um texto como excluído para indicar que deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que é importante.

Texto Sobrescrito

Para inserir coisas como x20 + 3.

Texto Subscrito

Para inserir coisas como H2O.
